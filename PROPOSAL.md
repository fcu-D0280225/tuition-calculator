# 補習班管理系統 — 業主提案

一套以「上課紀錄」為核心、能自動算學費與老師薪資、家長可一鍵收到清楚帳單的補習班專屬管理系統。

> 本文件為提供給補習班業主的功能提案，請搭配業務說明使用。技術細節請參考工程版 README。

---

## 一、系統功能總覽

| 模組 | 業主可以做什麼 |
|------|----------------|
| **學生管理** | 新增 / 編輯 / 刪除學生資料，掌握所有學生名冊。 |
| **老師管理** | 新增 / 編輯 / 刪除老師資料，方便排課與發薪。 |
| **課程設定** | 設定每堂家教課的學費與老師時薪；可加上「多人共學折扣率」自動計算優惠。 |
| **上課紀錄** | 老師當天打卡式輸入：學生、課程、時數、日期；介面只看得到教學欄位，金額自動帶入課程預設，避免在學生面前露出費率。 |
| **點名（團課）** | 團課透過 checkbox 一次點完出席名單，月底自動以「出席月份 × 月費」結算。 |
| **教材紀錄** | 學生領取教材自動加進當期帳單，業主不會漏收一本書錢。 |
| **學費計算** | 系統依日期區間自動分項加總：家教 + 團課 + 教材 = 學生本期應繳金額。 |
| **薪資結算** | 同一份報表也算出每位老師當期應領薪資（總時數 × 鐘點費），不必再用 Excel 對。 |
| **PDF 匯出** | 學費單 / 薪資單可一鍵下載 PDF，列印或寄給家長都方便。 |
| **帳單分享連結** | 每位學生可產生 30 天內有效的家長唯讀網址，家長免登入即可查看本期帳單。 |

## 二、核心亮點

### 1. 多人共學折扣公式（自動套用）

家教課可設定「每多一個共學學生，學費乘上 X%」。系統自動依當天同課同師實際出席人數計算折扣：

- 第 1 人：原價
- 第 2 人：原價 × 折扣率¹
- 第 3 人：原價 × 折扣率² ……依此類推

業主一次設好折扣率，之後不論哪一堂課有多少人共學，系統會自己算對。

### 2. 家長唯讀分享連結

不必開家長帳號，老師在結算頁點「分享」即可產生一條家長專屬網址。家長打開後看到的是：

- 學生姓名 + 期間
- 家教 / 團課 / 教材分項明細
- 應繳總額

連結預設 30 天有效，過期自動失效，無法編輯任何資料。

### 3. 多租戶隔離（規劃中・第二階段）

目前系統可一次服務一間補習班；下一階段將開放：

- 同一系統可給多間補習班共用，資料完全互不影響
- 同一間補習班內可邀請多位老師（共享資料但各自登入）
- 預設管理者帳號，老師帳號權限可細分

> 第一階段先回歸「最簡單登入模式」（單一管理員 / 助教），先穩定上線使用，再規劃多租戶。

## 四、技術規格

| 項目 | 說明 |
|------|------|
| **形式** | Web-based 系統，業主用瀏覽器即可使用，**無需安裝任何 App**。 |
| **手機支援** | 響應式設計，老師可用手機打開「點名頁」當行動 App 使用；亦支援加入桌面（PWA）。 |
| **資料儲存** | MySQL 8 資料庫，所有學生 / 老師 / 課程 / 帳單資料皆持久保存，可定期備份。 |
| **使用裝置** | 桌機、筆電、平板、手機皆可（建議使用 Chrome / Safari 最新版）。 |
| **資料安全** | 後端 API 全部要求登入；家長分享連結僅讀取單一學生的當期帳單，無法存取其他資料。 |
| **匯出格式** | 學費單 / 薪資單支援 PDF 匯出（A4 直印格式）。 |

## 六、常見問題（FAQ）

**Q1：我們的補習班沒有 IT 人員，能用嗎？**
A：可以。建議選擇「方案 B：託管於我方雲端」，您拿到網址就能直接使用，所有伺服器、備份、升級都由我方負責。

**Q2：要在每位老師的手機上裝 App 嗎？**
A：不用。系統是 Web-based，老師打開瀏覽器登入即可。如果常常使用點名功能，可選擇「加入主畫面」（PWA）讓圖示出現在手機桌面，使用時和原生 App 體驗相同。

**Q3：學生 / 課程資料能匯入嗎？**
A：可以。提供既有的 Excel / CSV，建置時會協助一次匯入；後續也可手動逐筆新增。

**Q4：家長收到的網址安全嗎？會看到別的學生的資料嗎？**
A：不會。每條網址只指向單一學生的特定期間帳單，且 30 天後自動失效。家長無法看到任何其他學生的資料，也無法修改任何內容。

**Q5：上課紀錄頁會被學生 / 家長看到嗎？金額不會曝光？**
A：上課紀錄頁專為老師教學打卡使用，已隱藏學費與老師時薪欄位，避免在學生面前不小心露出金額。後端依然完整記錄，結算時還是會以課程預設費率計算。

**Q6：多人共學折扣怎麼算？要每堂課手動修改嗎？**
A：不用。在「課程設定」中填一次折扣率即可（例如 90 代表每多一人乘 90%）。系統會自動偵測「同一天 / 同一堂課 / 同一位老師」上課的學生人數，並依此自動套用折扣，業主完全不必手動微調。

**Q7：結算可以一次跨好幾個月嗎？**
A：可以。日期區間自由選擇（一週、一月、一季、整學期皆可），系統會自動把該區間內的家教課、團課、教材費全部加總，並產出 PDF 帳單。

**Q8：如果家長要「年度繳一次」可以嗎？**
A：可以。結算頁日期選擇「整年」即可一次產出全年帳單。系統不限制週期。

**Q9：之後要新增功能（例如線上付款 / 簡訊通知）可以嗎？**
A：可以。本系統採模組化設計，新增功能屬客製化開發範圍，會另行報價。

**Q10：資料會備份嗎？要是出問題了怎麼辦？**
A：託管方案下我方每日自動備份資料庫；自架方案會在交付時提供備份指令範例。任何重大故障都有版本記錄可以還原。
